# models/

En este proyecto no entrenamos modelos supervisados (es un análisis de
datos exploratorio + correlacional, no de ML predictivo). La carpeta se
mantiene por la convención CCDS.

Si en el futuro se añaden modelos predictivos, van aquí con la misma
política de versionado: artefactos binarios grandes (`.pkl`, `.joblib`)
no se commitean; los scores y métricas sí.
